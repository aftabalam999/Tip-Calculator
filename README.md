# Tip Calculator

A simple and elegant web application to calculate tips per person and the total amount per person based on the bill amount, tip percentage, and number of people.

## Features

-   **Calculate Tip**: Easily calculate the tip amount per person.
-   **Total Per Person**: Calculate the total amount each person needs to pay.
-   **Input Validation**: Ensures all inputs (Bill Amount, Tip Percentage, Number of People) are positive numbers.
-   **Responsive Design**: Works seamlessly on desktops, tablets, and mobile devices.
-   **Reset Functionality**: Quickly reset all fields to start a new calculation.

## Technologies Used

-   **HTML5**: Structure of the webpage.
-   **CSS3**: Styling and responsive design.
-   **JavaScript**: Logic for calculations and DOM manipulation.

## How to Use

1.  Clone the repository or download the source code.
2.  Open `index.html` in your web browser.
3.  Enter the **Total Bill Amount**.
4.  Enter the **Tip Percentage**.
5.  Enter the **Total Number of People**.
6.  Click the **Calculate** button to see the results.
7.  Use the **Reset** button to clear the inputs.

## Project Structure

-   `index.html`: The main HTML file.
-   `style.css`: Contains all the styles for the application.
-   `app.js`: Contains the JavaScript logic for the calculator.
